# SplitWise Clone

SplitWise Clone is a modern, intuitive expense management application designed to help users track their spending and easily split bills with friends and groups. Built with speed and usability in mind, it provides a seamless financial overview.

## üöÄ Key Features

- **Personal Dashboard**: View your total balance, recent transactions, and financial summaries at a glance.
- **Expense Tracking**: Quickly add and categorize personal expenses.
- **Group Management**: Create groups for roommates, trips, or events.
- **Smart Bill Splitting**: Automatically calculate who owes what within a group.
- **Dynamic UI**: Responsive design with liquid layouts and smooth transitions.
- **Real-time Notifications**: Instant feedback using the Sonner notification library.

## üì¶ Installation & Setup

Follow these steps to get the project running locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/SplitWise-Clone.git
   cd SplitWise-Clone
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   The application will be available at `http://localhost:5173`.
